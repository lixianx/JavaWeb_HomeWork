# 1.项目主体结构
```
JavaWeb_HomeWork
├─src
│  ├─com
│  │  └─oa
│  │      └─learn
│  │          ├─bean        封装类
│  │          ├─filter      过滤器
│  │          ├─servlet     servlet
│  │          └─utils       封装工具类
│  └─resources              数据库和阿里云短信服务的配置
└─web
    ├─images                图标
    └─WEB-INF         
        └─lib               依赖
```
